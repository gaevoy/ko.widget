Overview

Big amount of rich UI functionality is always become difficult. There is Component Driven Development which aimed to manage code complexity. There are principles how to make components by Uncle Bob:

  • REP - The Release Reuse Equivalency Principle
  • CCP - The Common Closure Principle
  • CRP - The Common Reuse Principle
  • ADP - The Acyclic Dependencies Principle
  • SDP - The Stable Dependencies Principle
  • SAP - The Stable Abstractions Principle

ko.widget is JavaScript library which resolve code complexity during building rich UI in Component Driven Development way. It depends on KnockoutJS and RequireJS.
